05 · Locked Technology Decisions

Oracle for WST exchange rates?
CBS Sovereign Oracle Model — CBS is the oracle, not Chainlink
CBS is Samoa's monetary authority. They already publish WST rates. Routing CBS's own exchange rates through a foreign private oracle network is architecturally backwards and creates a foreign infrastructure dependency on sovereign monetary infrastructure. CBSOracle.sol — governed by Gnosis Safe 2-of-3, rate updates subject to TimelockController. Chainlink used only as secondary verification for international commodity data.
Current and future blockchain platform?
Phase 1–2: Polygon Amoy → Polygon Mainnet · Phase 3: Sovereign Geth Clique PoA · Phase 4: Sovereign compute network (technology-neutral)
Polygon for battle-tested EVM, largest audited library ecosystem (OZ), fastest path to research prototype. Sovereign chain for data residency and CBS validator control. Phase 4 sovereign compute eliminates all foreign infrastructure dependencies including oracle intermediaries via native HTTPS outcalls.
ZK Proofs?
Phase 3 — additive, ZKVerifier.sol on existing chain
Current NDIDS hash scheme is already ZK-compatible (commitment scheme). ZK proofs for privacy-preserving identity verification (prove attribute without revealing data) in Phase 3. Does not require rewriting existing contracts — additive layer only.
Oasis Protocol / Confidential EVM?
Phase 3–4 assessment — EVM-compatible migration path confirmed
Oasis Sapphire provides genuine on-chain confidentiality (TEE — even node operators cannot read encrypted state). Valuable for health and welfare records where "access-controlled" is not sufficient. TEE (Intel SGX) side-channel risks acknowledged. Network maturity risk at current scale. Revisit in Phase 3.
AI governance on sovereign infrastructure?
On-chain AI agent registry — AIAgentRegistry.sol (Phase 2)
Every AI agent issued a wallet address, registered in AIAgentRegistry with capability bitmask and expiry. All AI actions emit AIAgentAction event — permanently logged, distinguishable from human actions. Human-in-the-loop for high-value actions via TimelockController. Kill switch: CBS Governor revokes wallet address from all authorisedReaders.
One Maritime Window (OMW)?
Phase 2 — executeOMWClearance() on InteroperabilityHub, WST-DPI harbour dues
Samoa's January 2024 WTO TFA Article 10.4 obligation is unmet. Blueprint exists. DPI platform IS the architectural solution — InteroperabilityHub routes declarations to 6 ministry nodes (Customs, Port, Biosecurity, Health, Immigration, Maritime Finance). WST-DPI handles harbour dues. OMW module is commercially licensable to other Pacific Island nations.
Technology-neutral language?
MANDATORY in all client-facing content — no ICP / DFINITY / Canister / NNS / Chain Fusion / Motoko / Cycles
Sovereign compute network is described using: "threshold cryptography", "sovereign Wasm compute", "canister-style services", "HTTPS outcalls", "sovereign subnet". This protects commercial flexibility and keeps the research framing platform-neutral.
